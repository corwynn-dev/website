**Website:** [corwynn.dev](https://corwynn.dev)

## Products

- **GMS** - [greeklife.app](https://greeklife.app)
- **Penny Vault** - [pennyvault.app](https://pennyvault.app)

## Development

This is a static single-page site. To run locally, open `index.html` in a browser or use a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`

## Tech Stack

- HTML5
- CSS3 (custom properties, animations, responsive design)
- Google Fonts (Cabinet Grotesk, DM Sans)
- Formspree for contact form handling
